Calculator App - README
Overview
A simple and functional calculator application with a clean, intuitive interface for performing basic arithmetic operations.

Features
Basic Operations: Addition (+), Subtraction (-), Multiplication (*)

Number Input: Single digits (0-9), double zero (00) for quick entry

Special Functions:

AC: All Clear - resets the calculator completely

DE: Delete/Backspace - removes the last entered digit

=: Equals - calculates and displays the result

Responsive Layout: Buttons arranged in a logical, easy-to-use grid pattern

How to Use
Click number buttons to enter values

Use operation buttons (+, -, *) to select the desired calculation

Press "=" to see the result

Use "AC" to clear everything or "DE" to delete the last digit

Use "00" for quick entry of double zeros

Technical Details
Built with HTML, CSS, and JavaScript

Clean, user-friendly interface

Responsive design for various screen sizes

Simply open the index.html file in any modern web browser to start using the calculator.
